School Library Management System

Personal Project · Originally built for RIS, Kampala

A free, open-source, web-based library management system built with PHP and MySQL. Designed for schools that need a simple, reliable way to track books, borrowers, and returns without expensive software. Originally built for Redemption International School, Kampala.

  • Book catalog management — add, track copies, availability
  • Issue and return books with condition tracking to eliminate disputes
  • Automatic overdue detection with days-overdue count
  • Staff login authentication, live dashboard, and search by name or class
